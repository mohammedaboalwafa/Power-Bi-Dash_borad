# Power-Bi-Dash_borad
Power BI Dashboard Project for Data Analysis and Business Insights
Project Overview

In this project, I performed a full analytical cycle:

Data Cleaning & Transformation (Power Query)
Data Modeling (Star Schema Design)
DAX Calculations for KPIs & Metrics
Interactive Dashboard Design

The dashboard focuses on answering key business questions such as:

What are the top-performing products/categories?
Where are the main revenue drivers?
Which segments are underperforming?
How do trends change over time?
- Key Features
Dynamic & Interactive Dashboard (Filters, Slicers, Drill-down)
Clear KPI Cards for quick decision-making
Time-based analysis (Trends & Seasonality)
Comparative analysis between segments
Clean and user-friendly design
- Business Value

This dashboard helps stakeholders to:

Identify growth opportunities
Detect performance gaps
Optimize decision-making using real data
Understand patterns and trends
